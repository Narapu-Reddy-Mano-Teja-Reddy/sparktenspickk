# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Narapu-Reddy-Mano-Teja-Reddy/spark-tenspick.git
```

---

## Navigate to Project

```bash
cd spark-tenspick
```

---

## Run the Website

Simply open

```
index.html
```

or use VS Code Live Server.

---

## Using Live Server

Install Live Server Extension

Right Click

```
index.html
```

Click

```
Open with Live Server
```

Website will open automatically.

---

# ⚙ Firebase Configuration

Create

```
firebase-config.js
```

Add

```javascript
const firebaseConfig = {
 apiKey: "YOUR_API_KEY",
 authDomain: "YOUR_DOMAIN",
 projectId: "YOUR_PROJECT_ID",
 storageBucket: "YOUR_STORAGE_BUCKET",
 messagingSenderId: "YOUR_SENDER_ID",
 appId: "YOUR_APP_ID"
};
```
